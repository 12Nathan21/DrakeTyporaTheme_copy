## 介绍

安装 [Typora](https://typora.io/)


1. 截图使用的字体: [JetBrainsMono](https://www.jetbrains.com/lp/mono/), 安装即可
2. 代码高亮配色来自JetBrains Darcula
3. 脚注样式来自于Wiki百科





## 安装方式

复制`drake.css` 到`theme`目录下然后重启, 选择菜单栏主题

通过`设置 -> 外观 -> 打开主题文件夹`打开theme目录.
