最好的实时预览Markdown工具[Typora](https://typora.io/)使用的主题



1. 截图使用的字体: [JetBrainsMono](https://www.jetbrains.com/lp/mono/), 安装即可
2. 代码高亮配色来自JetBrains IDE



安装方式

复制`drake.css` 到`theme`目录下然后重启, 选择[菜单栏]()主题

通过`设置 -> 外观 -> 打开主题文件夹`打开theme目录.
